Install the local service

The OpenCode Session Manager browser extension needs a small local service to read your OpenCode database. This service runs only on your machine and does not send data anywhere.

1. Requirements

Node.js 18+ and sqlite3 are required.

2. Install on macOS / Linux

curl -fsSL https://raw.githubusercontent.com/Jiayixiao20/opencode-session-manager/main/install.sh | bash

This installs the CLI, registers it for auto-start, and starts the service.

3. Install on Windows

powershell -Command "& { Invoke-WebRequest -Uri https://raw.githubusercontent.com/Jiayixiao20/opencode-session-manager/main/install.ps1 -UseBasicParsing | Invoke-Expression }"

Windows auto-start support is coming soon. After install, run opencode-sessions serve in PowerShell.

4. Verify

Open http://localhost:8765 in your browser. If it loads, click the extension icon.

5. Custom database path?

OPENCODE_DB=/path/to/opencode.db opencode-sessions serve
